Note the the "cascading" nature of CSS

Even though we only styled the font of the body and not the p tag

The styles from the body "cascaded" down into all elements within it


But what if we want to style the same elements separately from each other?

For that we use either a class or id

Classes can be reused for multiple elements, id cannot

The order of precedence goes classes/id > innermost CSS > outermost CSS

So in this example: class > p > body


You can use a div tag to form a divison around several elements

You can use several classes on one single element, too

Or a class and an id

But not several ids at once
